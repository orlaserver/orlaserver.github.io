# Community

Orla is built for the community. Contributions are always welcome, whether it's reporting a bug, suggesting a feature, or writing code.

- [Report a bug or request a feature](https://github.com/harvard-cns/orla/issues)
- [How to contribute](https://github.com/harvard-cns/orla/blob/main/CONTRIBUTING.md)

If Orla is useful to you, consider [starring the project on GitHub](https://github.com/harvard-cns/orla). It helps others discover Orla and keeps the project going.

## Supporting Orla

If Orla becomes a tool you rely on, consider [sponsoring the project](https://github.com/sponsors/jadidbourbaki). Support helps with maintenance and future development.
