# orlaserver.github.io

GitHub Pages site for [orlaserver](https://github.com/orlaserver). Rendered from Markdown with [Jekyll](https://jekyllrb.com/) (built into GitHub Pages).

## Editing

- **Homepage:** edit `index.md`.
- **Styles:** edit `styles.css`.
- **Layout:** edit `_layouts/default.html` (wraps all pages).
- **New pages:** add any `*.md` with front matter `layout: default` and link to them (e.g. `[About](about.md)`).

No local build needed; GitHub Pages runs Jekyll when you push to `main`. The site is live at **https://orlaserver.github.io**.
